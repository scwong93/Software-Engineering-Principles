Calculate a tip:

IF bill total < $100 THEN
  COMPUTE total * 20%
ELSE
  COMPUTE total * 18%
END IF



Brushing teeth:

IF toothbrush has charge THEN
  apply toothpaste
  WHILE toothbrush is spinning
    move up and down
    move side to side
  END WHILE
  rinse toothbrush
ELSE
  recharge
END IF


Making Bagel:

IF bagel is available AND NOT in freezer THEN
  slice in half
  SET toaster to medium
  apply topping
ELSIF bagel is in freezer THEN
  SET microwave to defrost
  CALL make_bagel
ELSE
  find something else to eat
